# Appetite Link

### Laravel APIs + Vue Dashboard
Url: https://app.appetite.link

This web app was built following standard Laravel + Vue scaffolding with Inertia.js
https://laravel.com/docs/9.x/frontend

```
composer install
npm install
```

Build the assets

```
npm prod
```
